## 5. 代码示例

### 5.1 数据查询（SQL）

```sql
SELECT
    channel,
    COUNT(DISTINCT user_id)          AS new_users,
    ROUND(COUNT(DISTINCT user_id)
          * 100.0 / SUM(COUNT(DISTINCT user_id)) OVER(), 1) AS pct,
    ROUND(AVG(converted::int) * 100, 1) AS cvr
FROM acquisition_events
WHERE event_date BETWEEN '2025-01-01' AND '2025-12-31'
GROUP BY channel
ORDER BY new_users DESC;
```

### 5.2 留存率计算（Python）

```python
import pandas as pd

def retention_rate(df: pd.DataFrame, day: int) -> float:
    """计算第 N 日留存率"""
    cohort = df[df["install_date"] == df["install_date"].min()]
    retained = cohort[cohort["days_since_install"] == day]["user_id"].nunique()
    total = cohort["user_id"].nunique()
    return retained / total if total else 0.0

# 批量计算各日留存
for d in [1, 3, 7, 14, 30, 60, 90]:
    rate = retention_rate(events_df, day=d)
    print(f"D{d:>2} 留存率: {rate:.1%}")
```
